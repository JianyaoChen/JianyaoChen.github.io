# 陈健瑶

## 关于我

我是威斯康星大学密尔沃基分校信息研究学院信息科学博士候选人，目前正在寻求博士后研究机会。

我的博士研究聚焦在线健康社区中的 PTSD 相关讨论。基于混合研究方法，我考察个体如何在数字心理健康情境中表达情绪、寻求与提供社会支持、管理隐私相关的自我披露，并遭遇健康错误信息。方法上，我的研究结合计算文本分析与定量研究方法，包括大规模社交媒体数据采集、自然语言处理、深度学习、大语言模型与统计建模。

## 研究兴趣

- 数字健康
- 健康信息学
- 在线信息行为
- 在线社会与情感支持
- 数字福祉
- 数字平台中的隐私相关问题
